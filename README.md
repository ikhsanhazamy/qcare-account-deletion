# qcare-account-deletion

Pengguna aplikasi QCare dapat mengajukan penghapusan akun dengan langkah berikut:
Kirim email ke: support@qcare.app
Subjek: Penghapusan Akun QCare
Sertakan email yang terdaftar di aplikasi
Data yang dihapus:
Akun pengguna
Data profil
Riwayat antrean
Data yang disimpan:
Log transaksi (jika diwajibkan hukum)
Retensi data:
Data akan dihapus permanen maksimal 30 hari setelah permintaan diterima.
